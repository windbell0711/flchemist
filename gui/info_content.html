flchemist uses a plan-based workflow to batch-process files safely.

1. Draft (Create a Plan)

Choose an operation type and configure parameters. A .plan file is generated, containing a list of reversible actions (Copy, Move, Rename, Junction).

  • 按扩展名分类 — Sort files by extension into subdirectories
  • 按修改日期分类 — Sort files by modification date
  • 微信数据迁移 — Migrate WeChat data via NTFS Junction
  • AI 生成 — Describe what you want in natural language, and AI generates the plan

2. Open

Load an existing .plan file into the main window. The right panel shows the action list.

3. Execute

Run each action in sequence. On failure you can:
 • Abort — roll back the failed action only and stop
 • Rollback All — undo every completed action and stop

4. Revert

Reverse the currently loaded plan. Actions are undone in reverse order, restoring the original state.

Logs

Execution logs are saved to logs/ for auditing.
